# 贡献指南

感谢你考虑为 Ten-Bureau Framework 做贡献！以下是参与方式：

---

## 🐛 发现 Bug / 提需求

在 [Issues](https://github.com) 中创建新 issue：
- **Bug 报告**：描述问题、复现步骤、预期行为
- **功能请求**：描述你想加什么、为什么

---

## 💡 讨论

在 [Discussions](https://github.com) 中：
- 分享你使用框架的经验
- 提出改进建议
- 讨论学校治理方法论

---

## 🔧 提交代码

1. Fork 本仓库
2. 创建你的分支：`git checkout -b feature/my-feature`
3. 提交修改：`git commit -am 'Add new feature'`
4. 推送到分支：`git push origin feature/my-feature`
5. 创建 Pull Request

---

## 📝 贡献方向参考

- 新增部门的模板文件（如教务/德育/招生）
- 完善部署文档
- 编写使用案例（脱敏后）
- 翻译成英文
- 教师经验 Skill 化案例

---

## 行为准则

请保持开放、尊重、建设性的讨论氛围。
