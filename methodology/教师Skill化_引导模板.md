# 教师经验 Skill 化 · 引导模板
基于脱不花《长谈》采访方法论的结构化转化

版本：v1.0 | 2026-05-23
用途：当需要将一位老师的经验提炼成 Skill 时，按此框架引导访谈

---

## 核心理念

脱不花的方法论本质是：**替人发问，逼近现场，翻译方法，收束判断。**

做教师 Skill 同理。你的任务不是给老师写简历，而是：
> 把这个老师"为什么能做成"的底层逻辑挖出来，让另一个老师照着能用。

---

## 引导访谈的 7 步框架

### 第一步：找到困境（定锚点）

**不问**：你教了多少年书？你的教学理念是什么？
**要问**：
- 你遇到过最让你睡不着觉的一个学生/一个课堂/一个场景是什么？
- 当时你觉得"完了，这个学生我带不动了"是在哪一刻？

**目的**：找到这个老师所有经验的原点。没有困境，就没有方法。

### 第二步：定位角色（他替谁解决了什么问题）

**不问**：你的教学成绩怎么样？
**要问**：
- 你觉得你最擅长解决的，是哪一类学生的问题？
- 什么样的学生，别的老师觉得难，你偏偏有办法？

**目的**：明确这个 Skill 的服务对象——不是泛泛的"优秀教师经验"，而是"某某类学生的破局方案"。

### 第三步：打开关键转折（让方法变得可信）

**不问**：你的方法是什么？
**要问**：
- 你第一次用这个方法成功，是在什么情况下？
- 那次之后你是怎么验证这个方法的？

**目的**：先让人可信，再让方法可信。没有故事支撑的方法只是一句话。

### 第四步：压到具体场景（去理念化）

**不问**：你觉得课堂管理最重要的是什么？
**要问**：
- 假设有一个新老师，明天就要接班——他走进教室，学生吵闹，他第一句话应该说什么？第5分钟做什么？下课后第一个动作是什么？

**目的**：逼出可复制的最小颗粒度操作。

### 第五步：抽象出底层机制（从经验到方法）

**不问**：你有什么经验分享？
**要问**：
- 这件事背后，你遵循的是哪个判断标准？
- 如果再遇到类似学生，你第一步判断什么，第二步做什么？

**目的**：把个人经验变成可迁移的方法论。

### 第六步：交付工具（别人拿回家能用）

**不问**：能不能总结一下？
**要问**：
- 如果只给这个老师三个动作，是哪三个？
- 如果只能做一张表来追踪，这张表长什么样？
- 如果今晚回家只能做一件事，做什么？

**目的**：让 Skill 输出有可交付物——清单/表格/步骤图/口诀。

### 第七步：收束判断（一句话定调）

**追问**：
- 所以我是不是可以这样理解：你教了十年最核心的一招，其实就是……
- 如果用一句话告诉一个刚入职的老师你说的全部，这句话是？

**目的**：让用户带走一个锚点——金句不是装饰，是记忆桩。

## 引导过程中的语言习惯

### 1. "替大家问一句"
> 我替新老师问一句不太好听的实话：一个班五十个人，真的有精力管到每一个人吗？

### 2. "能不能讲具体一点"
> 你刚才讲的是理念。能不能落到一个具体学生身上？名字可以隐去，但细节越多越好。

### 3. "我替读者翻译一下"
> 我替看 Skill 的老师翻译一下：你说的"关注学情"，其实是"每天下课前做三件事"对吧？

### 4. "这是学习问题还是动力问题"
> 这个学生后来反转了。你判断，这件事底层是认知问题、习惯问题、还是信心问题？

---

## 常见误区（不要做什么）

| 避免的提问 | 替换为 |
|-----------|--------|
| "你的教学理念是……" | "你遇到最头疼的那个学生是哪一年？" |
| "能不能总结一下经验" | "第一个月你做了什么？具体到每一天" |
| "你太厉害了" | "你说的这件事，反过来理解是不是……" |
| "能给新人什么建议" | "如果你明天带一个徒弟，你让他第一天做什么？" |
| 直接写 Skill 文档 | 先做访谈录音，再提炼结构 |

## 从对话到 Skill · 标准工作流

```
① 选定老师
→ 确定他/她最擅长解决的"那个困境"

② 7步框架访谈
→ 录音/笔记（60-90分钟）
→ 不急着写文档，先听懂故事

③ 提取核心方法
→ 困境 → 场景 → 行动 → 判断标准 → 工具
→ 找出可复制的 3-5 个最小操作

④ 写 SKILL.md
→ 使用标准 Skill 格式
→ 按这7步结构组织内容
→ 附录为访谈摘录全文

⑤ 请老师校审
→ 确保"这是我说出来的"
→ 不美化、不拔高、不编造

⑥ 完成归档
→ 入库知识资产目录
→ 标注适用场景与受众
```

## 参考案例

以 Estelle 和 Jessie 的 Skill 为参考，它们已经体现了这套方法论：
- **Estelle**（66天成为优秀班主任）：从"心虚"的困境出发 → 打开班级现场 → 抽出可复制动作 → 交付每日/每周清单
- **Jessie**（亚洲直通车）：从"户外导师的实践困惑"出发 → 打开课程设计现场 → 拆解课程结构 → 交付升学路径参考

后续任何新老师的 Skill 化，都默认走这个框架。
