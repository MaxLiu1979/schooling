## 一、核心战略转向

**从**「拼顶尖升学成绩」 → **到**「学生出路诊断分析中心 + 确定性保录路径」

### 关键表述
> "学校其实更需要变成一个学生的诊断分析中心，给每个学生找一条适合的个人路径，人尽其用。"
> "只有大胆尝试差异化路线，学校才有可能活到下一个阶段。"
> "我们不能把一切都赌在学生成绩上，不确定性太高了。"

### 办学哲学更新
- 旧叙事：考名校 → 证明办学质量
- 新叙事：给每个基础普通的学生找到确定出口 → 现金流 + 口碑 + 差异化定位
- 教育方式：**"玩中带学"** — 给好的在校体验、比普高老师更理解学生、更好的成长经历

### 2. 雅思项目复盘结论
- ✅ 成功尝试，❌ 不成功产品
- 学生确实有提升，但**无前后测数据** → 未形成管理层可见结论
- 今年报名少，家长反馈：强度大、效果不明显、学生痛苦

### 3. 六一活动复盘
- 投入2000元 + 全体老师4-8天忙碌
- ROI不匹配，未形成可传播口碑
- 抢零食活动放在6.8万学费学校体验不合理

### 4. 剑桥升学路径判断
- 最低要求：数/物/高数三科两A一A*
- 现有学生几乎无人能达到 → **走此路无实际意义**

### 5. 亚洲班市场验证
- 苏州阿德科特亚洲班学费：4.9万/学期 → **10万/学期**（翻倍）
- 缴费截止：6月15日，最终到期7月底
- 验证：非名校升学需求群体体量很大

### 6. 外籍学生签证
- 需要金华教育局出具介绍信
- 统一回复口径：**"签证正在处理中"**
- 预留2天处理时间

## 四、各部门影响分析

### 国子监（教学）
- **最大冲击**：传统学科教学比重需重新定义
- 从"应试提分"转向"配合保录路径的能力培育"
- 雅思需重建为有前后测数据的可交付产品
- 需配套：新的课程评估体系、教师培训方向调整
- **风险**：一线教师能否理解并执行"玩中带学"？

### 礼部（品牌/招生）
- **重大利好**：确定性保录路径是差异化竞争利器
- 需重新包装前端内容（Kat 负责）
- 但须注意——对外口径不能滑向"交钱就能上大学"，需守住底线
- **对标竞品**：上海外国语大学下属学院做莫纳什保录已稳定运营十年

### 都察院（德育/学生管理）
- 新方向强调"好的在校体验"→ 德育工作从"管纪律"转向"建体验"
- 社团、活动、心理咨询需求可能上升
- 学生行为标准需重新对标（玩中带学≠放任自流）

### 户部（财务）
- 保录项目需要新的收费结构（学费+保录服务费？）
- 雅思/活动投入产出比需重新评估
- 亚洲班涨价（苏州案例）→ 融合部定价策略需重新考虑

### 宣传/招生（前端）
- 核心卖点从"考最好的大学"转向"给你确定的未来"
- 话术体系需重构
- 线索：浦江家长核心焦虑 = "读了三年不知道能不能上大学"

## 八、内阁判断

**这是一次勇敢且务实的战略转向。** 从内容看，Max 的思路是连贯的、基于现实的——剑桥数据否定了原有路径、苏州亚班涨价验证了市场需求、雅思项目复盘提供了产品交付的教训。

**最大挑战不在方向本身，而在执行中的两点：**
1. **合规底线**：保录路径如何包装才不越界（≠"交钱上大学"）
2. **向上管理**：卢阳如何理解这一转向——决定 Max 的职业安全感
3. **教师转型**：现有的教师团队是否有能力承接"诊断分析中心"的角色
