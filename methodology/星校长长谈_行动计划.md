# 星校长长谈 · 行动计划（模板版）
基于脱不花《长谈》采访风格拆解的方法论转化

版本：v1.0（模板版）
用途：将脱不花的十三步方法论转化为校长/教育管理者的公众表达与访谈框架

---

## 一、定位：不是"校长对话"，而是"教育翻译官"

脱不花是一个模式的轴心：**不是访问嘉宾，而是用嘉宾替普通人解决问题。**

"星校长长谈"同理：
- 不是"校长访谈名师/专家"
- 不是"学校领导对外发声"
- 而是：**用教育者的经验，替家长、老师、学生解决一个具体处境中的问题**

一句话定位：
> 星校长长谈：把教育问题问到现场，把学校方案讲到家长听得懂。

---

## 二、三步落地计划

### 第一步：选题储备（1-2周）

从以下三个来源积累困境清单：

**来源一：家长日常**
- 招生咨询中最高频的3个问题
- 家长会后的私下提问
- 家委会群里的真实焦虑

**来源二：教师一线**
- 班主任最难的3个教育瞬间
- 学科老师最被问住的场景
- 德育中最反复的问题

**来源三：学生现场**
- 学生真正困惑但不敢问的事
- 转学生/插班生的适应痛点
- 升学选择期的迷茫

> 工具：建一个「星校长长谈·选题池」笔记，随时收录

### 第二步：内容生产流程（每期标准流程）

借鉴脱不花的七步结构：

```
1️⃣ 困境定题
→ 找到家长/老师/学生真实卡住的地方
→ 不是A也不是B，第三条路在哪？

2️⃣ 嘉宾定位
→ 这位嘉宾能回答哪个具体问题？
→ 不是展示嘉宾多厉害，是请他解决一个问题

3️⃣ 打开人生
→ "您第一次意识到这个问题很严重，是在哪个具体现场？"
→ 先让人可信，再让方法可信

4️⃣ 压到场景
→ "不讲理念，假设有一个孩子……"
→ 把宏大议题压成一个人的具体处境

5️⃣ 抽出方法
→ 这背后是学习问题还是动力问题？
→ 底层机制是什么？

6️⃣ 交付工具
→ "如果只做一张表，应该看哪三项？"
→ "今晚回家只能做一件事，做什么？"

7️⃣ 收束判断
→ "所以我是不是可以这样理解……"
→ 帮观众带走一句能用的判断
```

### 第三步：输出形态规划

| 形态 | 频次 | 时长 | 定位 |
|------|------|------|------|
| 长谈视频/播客 | 双周/月 | 30-60min | 深度内容，积累品牌资产 |
| 短版金句切片 | 每期剪3-5条 | 1-2min | 社交媒体传播 |
| 文字稿精编 | 每期1篇 | 2000-3000字 | 公众号/家长群分发 |
| 工具卡片 | 每期1张 | A4可打印 | 家长拿回家能用 |

## 三、可立即使用的访谈模板

### 模板一：家长方向（邀请教育专家/同行）

> 今天我们不从概念开始，从一个真实问题开始。
>
> 很多家长现在卡在一个地方：一方面不想让孩子被单一分数压垮，另一方面又不敢真的放手。那教育到底有没有第三条路——既不放弃成绩，也不牺牲孩子的成长？
>
> 今天我想请您把这件事讲透。

### 模板二：教师方向（邀请一线老师）

> 我自己也是校长，也当过老师。有一件事我一直在想：规则要执行，孩子也要被看见——这个度到底在哪里？
>
> 您有没有遇到过这样一个学生：基础弱、习惯差、家长也放弃了，但您觉得还能救一救？

### 模板三：学生/家长故事方向（邀请家庭）

> 今天我们不谈理念，谈一个具体的家庭。您家孩子刚来学校的时候，是什么状态？
>
> 当时最让您睡不着觉的是什么问题？学校做了什么，让您觉得这条路还能走下去？

## 四、模仿时守住三个底线

1. **不要变成校长训话**
   → 脱不花的姿态是"我也在这个问题里"
   → 不是"我来告诉你"，而是"我替你去问"

2. **不要急着亮方案**
   → 先讲困境，再讲故事，再讲方法，最后才到学校
   → 顺序错了就像宣传

3. **不要只要金句**
   → 金句来自：真实处境 + 真实案例 + 真实方法
   → 只学句子不学结构，效果打折

## 五、与学校品牌的关系

"星校长长谈"不是一个独立的访谈节目，它是学校品牌表达的发动机：

| 内容方向 | 目标受众 | 品牌效果 |
|----------|----------|----------|
| 教育选择困境 | 焦虑的县域家庭 | 建立"懂我"的信任 |
| 成长故事 | 潜在家长 | 证明"孩子可以在这里变好" |
| 方法输出 | 同行/教育圈 | 建立专业权威 |
| 教师故事 | 内部团队 | 提升归属感与表达欲 |
