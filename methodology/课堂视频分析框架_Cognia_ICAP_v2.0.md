## 框架总览

本方法论基于**两套互补的国际标准**：

| 框架 | 视角 | 创始人/机构 | 核心问题 |
|------|------|------|------|
| **Cognia** | 教师行为 | Cognia Global（全球非营利教育认证组织） | "教师有没有做对的事？" |

## 第一部分：ICAP 认知参与框架

### ICAP 四层认知模式

> Interactive > Constructive > Active > Passive

| 模式 | 中文 | 定义 | 视频中可观察行为 |
|:--|------|------|------|
| **P**assive | 被动接收 | 学生仅接收信息，无外显行为 | 听讲时眼神呆滞 / 看PPT不记不画 / 安静但不一定在思考 |
| **A**ctive | 主动操作 | 学生对信息做物理操作 | 划线 / 抄板书 / 点按答题器 / 复述教师原话 |
| **C**onstructive | 建构生成 | 学生产出新的信息 | 用自己的话解释概念 / 做比较 / 画示意图 / 提问 / 类比 / 自问自答 |
| **I**nteractive | 交互共创 | 学生与同伴共同建构 | 小组辩论 / 互相纠错 / 接力回答 / 在同伴观点上补充 / 共写一篇总结 |

### ICAP 核心发现

- 认知深度每升一层，学习效果提升显著
- **关键区分**：Active 看学生手动没动；Constructive 看学生脑子有没有产出新东西
- 一堂好课中，P+A 不应超过 50%，C+I 至少占 30-40%
- 教师提问直接决定了学生处于哪一层：问"对不对"→ Passive；问"你怎么看"→ Constructive

### 视频分析时 ICAP 编码方法

1. 每2分钟为一个分析单元
2. 判定该单元中多数学生的ICAP层次
3. 统计全课 P/A/C/I 时间占比
4. 标注各模式切换的频率和触因

## 第二部分：Cognia 课堂观察标准

### 四大认证领域（30项标准总纲）

| 领域 | 标准数 | 视频分析聚焦 |
|------|:---:|------|
| 学习文化 Culture of Learning | 6 | 尊重、包容、无偏见 |
| 学习领导力 Leadership for Learning | 9 | 目标清晰、反馈机制 |
| 学习参与度 Engagement of Learning | 8 | 学生主动程度（对接ICAP） |
| 学习成长 Growth in Learning | 7 | 产出质量、思维递进 |

### 教师观察五维度（23项指标）

**维度1：文化氛围（4项）**
A1 安全包容 / A2 高期望 / A3 互相尊重 / A4 有序环境

**维度2：学习（5项）**
B1 目标清晰 / B2 策略多样 / B3 检查理解 / B4 高阶思维 / B5 及时反馈

**维度3：基本要素（5项）**
C1 学科知识扎实 / C2 学科语言准确 / C3 对标课标 / C4 资源得当 / C5 时间管理

**维度4：自主性（5项）**
D1 赋权学生 / D2 鼓励提问 / D3 提供选择 / D4 培养自学 / D5 促进反思

**维度5：关系（4项）**
E1 师生关系 / E2 个体关注 / E3 同伴合作 / E4 家校连接

## 第三部分：双框架融合分析工作流

```
课堂录像（30-45分钟）
  ↓
▲ 第1层 · 时间轴切分
   → 每2分钟一个分析单元，标记教学活动类型
  ↓
▲ 第2层 · ICAP学生认知编码
   → 逐单元判定 P/A/C/I 层次
   → 统计四层占比、切换频率、升降趋势
  ↓
▲ 第3层 · Cognia教师行为评分
   → 按23项指标逐项打分（1-4分制）
   → 标注视频时间码作为评分证据
  ↓
▲ 第4层 · 交叉分析
   → ICAP的C+I比例 vs Cognia的B4高阶思维评分
   → ICAP的P过高 vs Cognia的B2教学策略评分
   → 数据矛盾 = 课堂的核心问题所在
  ↓
▲ 第5层 · 输出报告
   → Part A: 课堂认知参与热力图（ICAP时间线）
   → Part B: 教师行为雷达图（Cognia五维评分）
   → Part C: 关键发现 + 可操作改进建议
```
