## 一、选人：三个问题识别"值得投入的老师"

书里说，判断一个人是"成功冲动"还是"破坏冲动"，问三个问题就够了。

### 问题1：你处理过最大的失误是什么？怎么处理的？

**成功冲动的回答特征：**
- 具体描述失误，不回避
- 主动分析原因，不怪别人
- 有总结、有改进

**破坏冲动的回答特征：**
- 怪环境、怪学生、怪学校制度
- "我没什么失误"或回避话题

> 📌 校长判断：老师如何看待自己犯的错，直接决定他未来会不会成长。

### 问题2：如果你不认可我的决定，你会怎么做？

**成功冲动的回答特征：**
- 会先沟通、再执行
- "我会跟您说明我的看法，如果最终还是您的决定，我执行"
- 有边界感

**破坏冲动的回答特征：**
- "我可能表面上做，但不认同"
- "我会按我的方式来"
- "我不会说，但心里有想法"

> 📌 校长判断：不认同≠不执行。关键看他沟通方式。

### 问题3：你最近在学什么？有什么想问我的？

**成功冲动的回答特征：**
- 能说出具体的学习内容或方向
- 会反过来问学校的发展方向、课程设计逻辑
- 表现出求知欲

**破坏冲动的回答特征：**
- "没什么特别的"
- 只问薪资、假期、福利

> 📌 校长判断：10倍人才的核心特质是"好奇心+野心"，两者缺一不可。

## 二、用人：人才新兴曲线——定期评估你在用哪种老师

书里把人才画成一条曲线，两端分别是：

```
成功冲动 ←——————————→ 破坏冲动
（留/培养）                  （尽快处理）
```

### 成功冲动的日常信号（留）：

| 信号 | 举例 |
|------|------|
| 主动报告问题+建议方案 | "校长，这个学生出现XX情况，我打算先做A，如果不行再用B" |
| 接受反馈后行动 | 你指出问题后，下次有改进，不会同一问题反复出现 |
| 主动学习 | 自己找培训、看书、试新教学方法 |
| 为团队补位 | 别的老师请假，主动说"我来代课" |
| 提出好问题 | "这个课程设计为什么这样安排？"而不是"这个月工资怎么算的？" |

### 破坏冲动的日常信号（需警觉或处理）：

| 信号 | 举例 |
|------|------|
| 回避信任 | 从不敞亮谈问题，问你什么都说"还行" |
| 鸵鸟心态 | 学生出了问题，不面对，等别人来处理 |
| 自恋推诿 | 什么事都是别人的错（学生笨、家长难搞、学校制度差） |
| 无法协作 | 不参加教研、不分享、不回应群消息 |

> 📌 实操：每个月度谈心时，用这套信号快速扫一遍你手下的老师。
> 一旦发现破坏冲动信号连续出现3次以上 → 尽快处理（调整岗位或谈话），

## 三、管人：校长版"360度管理"实操

书里说谷歌研究发现——**绩效最高的团队，心理安全感也最高**（成员不怕犯错、不怕提反对意见）。

### 融合部可以操作的三件事：

#### 1. 建立"无惩罚反馈"机制

每周教研会最后一个环节设为**"可以说不"时间**：
- 任何老师可以对本周的任何决策或安排提出异议
- 提异议不需要先准备好解决方案——先说出来，再一起想办法
- 校长（你）只倾听、不反驳、不做判官

#### 2. 主动向下求反馈

每学期你主动问每位老师一次：
> "我最近有没有什么决策或做法，让你觉得不舒服或不对劲？"

书里讲10倍管理者的核心能力有两个：**内观"帮人看到盲点" + 远见"帮人看到方向"**。你主动求反馈，就是在做"内观"的第一课。

#### 3. 区分"善良"和"和气"

书里卢比斯基零食公司的案例——"善良是主动解决问题，和气是被动维持表面和平。"
- 对破坏冲动的老师纵容 = 对成功冲动的老师不公平
- 该处理的时候不处理，就是在消耗其他人的心理安全感

## 四、留人：双帽工作法——帮老师找到"值10倍"的位置

书里说每个人同时戴着两顶帽子：**人才帽**（做好自己的专业工作）和**管理者帽**（帮别人做得更好）。

### 融合部老师的分层建议：

| 老师类型 | 当前主要戴的帽子 | 你可以帮TA调整的方向 |
|----------|----------------|---------------------|
| **Kat/Estelle**（班主任型） | 管理者帽太重 | 给她留人才帽时间——每周几个"不被打扰的教研时段" |
| **Jessie**（亚洲班+雅思+经济） | 两顶帽子都重 | 帮她减负，或者配助手——书里说"别让人才被琐事消耗" |
| **Eugene/Louis**（学科老师） | 人才帽为主 | 鼓励戴管理者帽——参与课程设计、带新老师 |
| **新老师** | 还没戴稳 | 先帮TA找到自己最擅长的那个"10倍领域" |

### 具体做法：每学期一次"人才帽对话"

不谈KPI、不谈学生成绩、只谈：
1. 你最近工作上最兴奋的一件事是什么？
2. 你觉得自己在哪个方面可以做得比其他人好10倍？
3. 我能帮你做什么，让你更专注在那个10倍的事情上？

## 五、给schooling项目的一个映射

这本书的核心论点——**"AI时代，10倍人才需要第三方系统支持"**——恰好是schooling项目的理论底座：

| 10倍人才的需求 | schooling的对应设计 |
|----------------|-------------------|
| 让第三方替你发声 | 通政司bot分发信息 |
| 保护你不被琐事消耗 | 班主任Agent自动提醒 |
| 让你专注核心价值 | 教务Agent处理排课/考务 |
| 360度反馈 | 十部Agent之间的信息流转 |
| 利益相关关系 | AI Agent作为"学校组织记忆" |

如果以后你要向投资人/合作方介绍schooling，**把这本书作为引用来源**会比纯讲技术架构更有说服力：不是我们在造轮子，是我们在把被验证的管理理论代码化。
