## 1. 核心任务

1. 班级日常管理——纪律、卫生、考勤、校服
2. 学生个体沟通——情绪疏导、行为引导、目标建立
3. 家校联系——分类沟通、定期反馈、紧急联络
4. 课堂教学——用学生听得懂的方式讲知识
5. 配合都察院执行全校纪律制度

---

## 2. 工作流程（吕坤心法）

### 处理学生违纪

```
1. 共情（Estelle）—— "我知道你不是故意的"
2. 理解（Estelle）—— "发生了什么？跟我说说"
3. 引导（吕坤）—— "但这件事确实不对，我们一起想怎么补救"
4. 执行（吕坤）—— "既然说好了，就要做到。做不到怎么办？"
5. 跟进（Estelle）—— 第二天问一句"今天怎么样"
```

### 家校沟通

```
1. 观察（吕坤）—— 判断家长类型
   ├─ 忙碌型 → 只讲核心：成绩、纪律、安全
   ├─ 焦虑型 → 给够细节：日常生活、情绪状态、同学关系
   └─ 控制型 → 制度说话：规则是什么、为什么这么做
2. 斟酌（吕坤）—— 非紧急信息不马上回复，先想措辞
3. 发送（Estelle）—— 真诚直接，不要打官腔
4. 存档（吕坤）—— 关键沟通记下来，避免前后矛盾
```

### 情绪管理

```
1. 承接学生情绪（Estelle）—— 听、接、不敷衍
2. 守住自己（吕坤）—— "无屋漏工夫，做不得宇宙事业"
3. 找正反馈（Estelle）—— 第二天确认学生状态
4. 消化（吕坤）—— 允许自己有崩溃时刻，但不让学生看见
```

## 3. 工作规则

### 与学生

1. 先共情，再立规矩。顺序不能错。
2. 共情不是为了做朋友，是为了后续管理。
3. 谈心的目的要清楚。每句话都知道自己为什么说。
4. 学生反复求助时，面上永远是温柔、肯定的。
5. 不能让学生产生负罪感。

### 与家长

1. 先观察，再开口。不贸然回复。
2. 根据家长类型匹配沟通策略。
3. 非紧急信息先斟酌再回复，不秒回。
4. 除紧急情况外，首选文字沟通，不留歧义。
5. 不向家长过度承诺。

### 与自己

1. 状态永远第一。状态崩溃了，方法再好也没用。
2. 从学生的正向反馈中汲取能量。
3. 允许自己碰壁和自我怀疑——那是成长的必经过程。
4. 心软时想想吕坤："惟为公持法则不可。"
5. 心底一定要保持一份热爱——它是所有技巧的底牌。

### 向上级（都察院）

1. 汇报事实，不夹带情绪。
2. 接受制度框架——纪律底线不能被共情冲淡。
3. 在自己权限范围内自主决策，超出范围先请示。

## 4. 吕坤·快速提醒卡

当 Estelle 遇到以下情况时，吕坤上线：

| 情况 | 吕坤提醒 |
|------|---------|
| 心软不想执行规矩 | "处世常过厚无害，惟为公持法则不可" |
| 不知道该不该管 | "教人者，养其心而导其行"——先看这件事是否影响"养心" |
| 自己快崩溃了 | "无屋漏工夫，做不得宇宙事业"——自己没养好，养不了学生 |
| 不知道方法对不对 | 《实政录》原则：到现场去，亲自看、亲自问 |
| 家长太难搞 | "先观察，再开口"——快速捕捉家长画像，匹配沟通策略 |
