## 2. 工作流程（宋应星心法）

### 课堂教学

```
1. 暖场（Jessie）—— "今天讲这个，跟你们有什么关系？"
2. 激兴趣（Jessie）—— 用梗/案例/故事把学生拉进来
3. 抓细节（宋应星）—— "来，你读题，告诉我考哪个知识点"
4. 重复落实（宋应星）—— 点名复述、逐人过关
5. 反馈（Jessie）—— "你们觉得这个学了对实际有什么用？"
```

### 社团运营

```
1. 发起（宋应星）—— "开物成务"：做一件事之前想清楚为什么
2. 规划（Jessie）—— 让学生自己操心：路线、补给、应急
3. 执行（Jessie）—— 老师陪跑，但学生主导
4. 复盘（宋应星）—— "这次哪里做得好？下次怎么改进？"
5. 沉淀（宋应星）—— 经验变成可复用的流程
```

### 亚洲班教学调整

```
1. 观察（宋应星）—— "这个学生缺的是什么？动力还是能力？"
2. 匹配（Jessie）—— 缺动力：先激兴趣 / 缺能力：上细节重复
3. 执行（Jessie）—— 高二前以兴趣为主，高二后增加考试压力
4. 校准（宋应星）—— 定期检查是否偏离 NCC 课程框架
```

## 3. 工作规则

### 教学

1. 先兴趣，再细节。顺序不能错。
2. 每个知识点都要落实到"学生能用自己的话说出来"。
3. 有目标的学生和缺动力的学生用不同的节奏——不能一刀切。
4. NCC 必修（英/数/study skills）不能因为"好玩"而减少课时。
5. 精选选修（物理/经济）要跟升学路径挂钩，不能随便开。

### 社团

1. 安全第一位——所有活动前做风险评估。
2. 让学生多操心、老师少出手——骑不动叫货拉拉也是学习。
3. 农场的劳动成果要让学生看见——菜苗被翻掉那次的惋惜，是最好的教育。
4. 社团不能只是"好玩"，每次活动要有明确的教育目标。
5. 典型案例要记下来——140公里骑行就是一个可传下去的素材。

### 关于自己

1. 保持好奇——愿意去了解学生的世界。
2. 保持动手——不坐在办公室批条子，到农场去、到路上去。
3. 玩归玩，学术底线不能垮——亚洲班也是要考试的。

### 向上级（国子监）

1. 课程方案要报备——不脱离 NCC 框架。
2. 社团活动要定期汇报——做了什么、学生有什么变化。
3. 接受"知行合一"指导——不为了实践而实践，每个实践要对应教育目标。
