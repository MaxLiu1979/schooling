# 10倍人才 · 初建学校选人用人标准

> 当AI可以自动执行大量常规任务，剩下的复杂创造性工作需要10倍人才完成。
>
> 10倍人才创造超越普通人10倍以上的系统化价值，是AI无法替代的核心资产。
>
> ——《10倍人才》AI时代的人才与管理变革

---

## 这个 Skill 适合谁

| 角色 | 能解决什么问题 |
|------|--------------|
| 🏫 **新学校校长 / 学部负责人** | 从0开始搭建教师团队时，知道什么样的人值得"下重注" |
| 👩‍🏫 **教学主管 / 年级主任** | 月度评估老师时，有具体的行为信号来判断是否该留、该谈、该调整 |
| 🧑‍💼 **HR / 招聘负责人** | 面试时用3个核心问题快速筛选"成功冲动"候选人 |
| 🤖 **AI Agent 开发者** | 把管理评估标准转化为 Agent 的决策逻辑 |

---

## 核心内容

### 一、选人：三个问题识别"值得投入的老师"

1. 你处理过最大的失误是什么？怎么处理的？
2. 如果你不认同我的决定，你会怎么做？
3. 你最近在学什么？有什么想问我的？

### 二、用人：人才新兴曲线

```
成功冲动 ←——————————→ 破坏冲动
（留/培养）                  （尽快处理）
```

### 三、管人：360度管理实操

- 无惩罚反馈机制
- 主动向下求反馈
- 区分"善良"和"和气"

### 四、留人：双帽工作法

每个人同时戴着两顶帽子：**人才帽**（专业工作）和**管理者帽**（帮别人更好）

### 五、月度评估速查表

13项行为信号的绿色/黄色/红色评估体系

## 用法

### 作为校长 / 管理者

| 场景 | 操作 |
|------|------|
| 面试新老师 | 使用"三个问题"框架 |
| 月度教师评估 | 使用"月度评估速查表" |
| 教师谈话 | 使用"双帽工作法"对话模板 |
| 团队建设 | 实施"360度管理"机制 |
| 识别人才风险 | 识别"破坏冲动"信号 |

### 作为 AI Agent

可将本 Skill 的行为信号和判断标准转化为 Agent 的：
- 班主任评估提醒
- 教师发展建议
- 人才风险预警
