# Schooling

> 让学校真实发生的工作，慢慢长成组织记忆。  
> Let real school work slowly grow into organizational memory.

---

## 这是什么｜What is this

Schooling 是一个诞生于真实高中现场的 AI 工作流实验项目。  

Schooling is an experimental AI workflow project built inside a real high school.

它不是为了替代老师。  
It is not designed to replace teachers.

它是为了帮助学校：

It is designed to help schools:

- 留下真实的学生支持工作  
  Preserve real student support work

- 整理散落的沟通与记录  
  Organize scattered communication and records

- 支持班主任长期跟进学生  
  Support long-term homeroom teacher follow-up

- 让“一生一方案”慢慢长出来  
  Slowly grow “one student, one evolving support plan”

---

# 学校真正的问题｜Real School Problems

学校从来不缺努力。  
Schools do not lack effort.

学校真正缺的是：

What schools truly lack are:

- 组织记忆  
  Organizational memory

- 长期连续的学生支持  
  Long-term continuity in student support

- 可继承的班主任经验  
  Reusable homeroom teacher experience

- 可持续协同的工作流  
  Sustainable collaboration workflows

很多学校里：

In many schools:

- 家校沟通散落在微信里  
  Conversations disappear in WeChat

- 班主任经验无法沉淀  
  Homeroom teacher experience cannot be inherited

- 学生支持工作高度碎片化  
  Student support becomes fragmented

- 一旦人员变化，一切重新开始  
  Everything restarts when people change
